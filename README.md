A game using ReactJS + SVG. Our cannon will shoot aliens (UFO)

### Prerequisites
Knowledge on ReactJS, SVG, NodeJS 8 (parcel may not work with NodeJS 6,7)

### Techniques
We use Parcel, React, SVG
